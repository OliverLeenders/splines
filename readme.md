# Didactic & Visual B-Spline Editor

This is a minimal editor to create and manipulate cubic B-Splines. It was created with the intention of showing directly and visually how cubic B-Splines function and give an intuition about what is happening.

## Features

* Double-Click to create a control point.
* Double-Click an existing control point to delete it.
* Drag the control points directly affects the spline.
